Elementor widget: default image, hover image, overlay with heading and rich text.

== Installation ==

1. Zip the `image-hover-elementor` folder or upload it to `/wp-content/plugins/`.
2. Activate the plugin in WordPress.
3. Make sure Elementor (or Elementor Pro) is active.
4. Edit a page with Elementor → search **Image Hover** in the widget panel.
5. Add under category **Image Hover** or **Basic**.

== Usage ==

**Content → Image**
- Choose Image (normal state)
- Hover Image (shows on mouse hover)
- Image size, caption, link, lightbox (same as Elementor Image)

**Content → Hover Overlay**
- Show Overlay on Hover
- Heading + HTML tag
- Rich Text (WYSIWYG editor)

**Style**
- Image: alignment, width, height, border, shadow, CSS filters
- Overlay: background color, padding, content position
- Heading & Rich Text typography and colors
